# Camera Terminology Reference

## Shot Types

| Shot Type | Description | Use Case |
|-----------|-------------|----------|
| Extreme Wide Shot (EWS) | Full environment, subject tiny | Establishing shots |
| Wide Shot (WS) | Full figure, environment visible | Scene setting |
| Medium Shot (MS) | Waist up | Dialogue scenes |
| Close-Up (CU) | Face fills frame | Emotion focus |
| Extreme Close-Up (ECU) | Single feature (eyes, hands) | Intensity |
| Over-the-Shoulder (OTS) | Behind one person, facing another | Dialogue |

## Camera Movements

- **Pan**: Horizontal rotation on fixed point
- **Tilt**: Vertical rotation on fixed point
- **Dolly**: Camera moves toward/away from subject
- **Tracking**: Camera follows subject movement
- **Crane**: Elevated camera movement
- **Zoom**: Focal length change (different from dolly)

## Composition Rules

- **Rule of Thirds**: Key elements at 1/3 or 2/3 points
- **Headroom**: Space above character's head
- **Leadroom**: Space in front of subject's gaze/direction
- **Frame Within Frame**: Use doorways, windows
